# evolve — bug-shaped intake

**Status:** designed.

A bug or feature request is a small idea entering through a different door: RCA +
fix-design is a refinement-and-design activity (human-gated, like Design); the fix
itself runs the same inner loop with the same gates. Because brownfield is
first-class, this path is a peer of greenfield, not a lesser one. Operate turns
production signal into these intakes.

## Acceptance

- A bug intake runs the engine with the same gates; RCA + fix-design is
  human-approved.
